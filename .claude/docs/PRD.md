# 富禮分會常駐營運平台（bni-tool）- 產品需求文件 (PRD)

> 由 `/concept` 維護

## 專案概述

### 專案目標

打造 BNI 富禮分會的**常駐營運工具**——不綁屆期、跨屆累積資產。以「一對一約訪引擎」為核心，搭配會員名錄與 PALMS 數據儀表板，解決三個問題：

1. **約訪難**：成員間一對一約時間來回拉扯成本高（核心，我們的差異化）
2. **數據不行動**：PALMS 數字躺在報表裡，成員不知道「自己差什麼、該找誰」
3. **名錄分散**：會員專業別、產業鏈關係、25 秒順序沒有好查的地方

競品參考：富樂分會「富樂 LIVE」（分析見 `ANALYSIS-FULE-LIVE.md`）。它驗證了 BNI 文化中儀表板與遊戲化的黏著力，但屆期綁定、維運靠一人手改檔案、個資與密碼公開裸奔。我們的定位互補且修正這些坑：**它記錄行為，我們促成行為**。

### 核心價值

1. **促成閉環**：從「預測綠燈告訴你差一場一對一」到「點下去直接發預約」，數據直通行動
2. **資料不過期**：混合制空檔登記（固定模式＋一次性＋例外），維護成本極低
3. **常駐架構**：資料進資料庫、權限用 RLS、PALMS 用上傳匯入——不依賴任何單一維護者手改程式

## 目標用戶

### 用戶畫像

- BNI 富禮分會成員（約 20–50 人），中小企業主／業務／專業人士
- 重度 LINE 使用者，3C 熟練度落差大 → 介面極簡、手機優先
- 分會幹部（副主席／秘書）：每週從中心區系統下載 PALMS Excel，負責匯入

### 使用場景

1. **設定空檔（低頻）**：每週固定模式設定一次；行程變動時加開／挖除，全部可隨時修改
2. **發起／回應預約（高頻）**：看到對方可約時段 → 送出請求（附留言）→ 對方確認／提議改期／婉拒；已成立可改期取消，雙方收 LINE 通知
3. **找該約的人**：一對一矩陣看「這季還沒約過誰」；預測綠燈看「我差什麼就能轉綠」
4. **查人**：產業服務鏈名錄找「做防水的是哪位」；查 25 秒順序
5. **幹部匯入（每週）**：上傳 PALMS Excel → 紅綠燈表、預測綠燈自動更新
6. **完成記錄**：一對一結束打卡，矩陣更新，PALMS 回報有依據

## 功能需求

### 核心功能 (MVP)

- [ ] **成員與登入**
  - LINE Login（免密碼，綁定推播身分）；邀請制，管理員預建名單
  - 成員資料：姓名、公司、專業別、頭像、**產業鏈分類、25 秒順序**
- [ ] **會員名錄＋產業服務鏈**
  - 按產業鏈分類瀏覽（參考富樂 DirPage），搜尋，25 秒順序表
- [ ] **空檔管理（混合制，全部可隨時修改）**
  - 每週固定模式＋一次性加開＋例外挖除
- [ ] **預約流程（請求→確認制）**
  - 瀏覽可約時段（固定＋一次性−例外−已占用）→ 請求（附留言）→ 確認／改期／婉拒
  - 已成立預約可改期、取消，雙方通知
- [ ] **LINE 通知**（Messaging API 推播；新請求、確認、改期、取消、前一日提醒）
- [ ] **一對一矩陣**
  - 季度視圖（約過／進行中／沒約過）、完成打卡、從矩陣直接發起預約

### 第二優先（PALMS 數據模組，緊接 MVP）

- [ ] **PALMS 匯入後台**：幹部上傳中心區 Excel → 解析入庫（每週快照，保留歷史）
- [ ] **紅綠燈表**：歷月各成員燈號與分數明細（依 BNI 中心區計分規則）
- [ ] **預測綠燈＋補救名單**：試算「再 N 位達標」；列出「補一場一對一／一張引薦單就轉綠」的成員——**名單上的人可直接點擊發起預約**（富樂做不到的閉環）
- [ ] **培訓率追蹤**（輕量：場次登記＋參加率）

### 次要功能

- [ ] 配對建議、個人統計、PALMS 回報輔助匯出
- [ ] 管理員後台：成員邀請／停用、產業鏈分類維護、25 秒順序調整
- [ ] 目標計算機（年營業額反推每週一對一次數，靜態小工具）

### 未來考慮（設計預留、暫不實作）

- **賽季競賽模組**：小組計分、排行榜、戰報跑馬燈、光榮榜。資料模型預留 `seasons`（賽季）維度，賽季結束歸檔不刪；富禮確定舉辦競賽時再開發
- 留言板（先觀察；若做採具名制）
- Google Calendar 同步、多分會支援

## 非功能需求

### 效能要求
- 單分會規模（≤ 50 人＋幹部），免費方案可支撐；手機開啟到可操作 < 3 秒

### 安全要求（吸取富樂 LIVE 教訓，一票否決級）
- **禁止**前端硬編碼任何密碼／金鑰（富樂把所有角色密碼放在公開 HTML）
- **禁止** PALMS 個資公開可讀（富樂 85 人出席率／引薦數任何人可看）——所有資料 RLS 限登入成員
- 身分用 LINE Login，權限分級：成員／幹部（PALMS 匯入、名單管理）
- Repo public，程式碼與資料嚴格分離；`.env` 不進版控

### 相容性
- 手機優先（LINE in-app browser 必須正常）、支援桌面

### 通知用量
- LINE 免費 200 則/月恐不足：合併通知（單場預約全流程 ≤ 3 則）、每日提醒合併、用量監控；必要時分會付費升級
